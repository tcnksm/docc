# docc(1)

`docc` open your project documents.

## Usage

By default, `docc` will open your project page by browser.

```bash
$ docc
$ docc /path/to/project
```

You can open README.md file by your `$EDITOR`.

```bash
$ docc -e
$ docc -e /path/to/project
```

## Install

```bash
$ go get github.com/tcnksm/docc
```
